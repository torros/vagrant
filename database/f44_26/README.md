# Oracle AI Database 26ai on Fedora 44

A simple Vagrant build for Oracle AI Database 26ai on Fedora 44.

## Required Software

* [Vagrant](https://www.vagrantup.com/downloads.html)
* [VirtualBox](https://www.virtualbox.org/wiki/Downloads)
* [Oracle Database](https://www.oracle.com/technetwork/database/enterprise-edition/downloads/oracle26ai-linux-5462157.html)

Place the Oracle database software in the "software" directory before calling the `vagrant up` command.
